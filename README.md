
# Sentence-Similarity-Web

## 项目简介

本项目提供一个基于 Flask + 预训练句子编码器（如 SBERT）的在线句子余弦相似度比较工具。

---

## 原理简介

### 句子编码器

- 采用 Huggingface 生态下的预训练句子编码器（如 SBERT、MiniLM、DistilUSE）。
- 作用是把任意自然语言句子转换为一个稠密向量（向量长度通常为384或768维）。
- 句子的语义会被编码进其向量中。






### 句子编码器

- **定义**：句子编码器是一种深度学习模型，可以自动将自然语言中的整个句子映射（编码）为一个定长、稠密的数值向量（如384维或768维），这个向量包含了该句子的语义和主要信息。
- **主流模型**：本项目使用 Huggingface 生态下的预训练句子编码器，如 SBERT（Sentence-BERT）、MiniLM、DistilUSE 等。
- **编码方式**：
  - 编码器首先将句子分词，嵌入为词向量，然后通过多层 Transformer 神经网络进行语境理解建模。
  - 模型将整个句子建模成一个整体，提取其上下文信息和语义特征，最终输出一个向量。
- **稠密表示**：输出向量通常是高维的实数数组（如 `[0.23, -0.12, ...]`），代表该句子的压缩语义特征，维数固定（与句子长度无关）。
- **优势**：
  - 句子的语义（如主旨、情感、核心信息）被聚合进输出向量。
  - 不同但语义接近的句子，会被编码为相近的向量。例如，“今天天气很好”和“今天阳光明媚”经过编码后，向量之间的距离较小。
  - 为了衡量两个句子在语义上的相似程度，可以直接比较它们各自的向量。

### 余弦相似度

- 两个句子分别获得各自的向量表示后，用余弦相似度（cosine similarity）进行比较。
- 公式为：  
  CosineSim(A, B) = (A · B) / (||A|| × ||B||)
- 值范围为 -1 ~ 1，数值越大说明语义越接近。

**工作流程：**
1. 输入两个句子。
2. 分别编码为向量。
3. 计算二者余弦相似度，输出得分。

## 安装与使用

1. 安装依赖
    ```bash
    pip install flask torch transformers
    ```
2. 运行服务
    ```bash
    python app.py
    ```
3. 打开浏览器访问 [http://127.0.0.1:5000](http://127.0.0.1:5000)，输入需要比较的两个句子，即可获得相似度结果。
